# 📝 Quick Notes App

A modern and responsive Note-Taking Application built using **Vanilla JavaScript**, HTML, and CSS.  
This project allows users to create, edit, delete, and save notes directly in the browser using **localStorage** for persistent data storage.

The application also includes a clean responsive UI, hover actions, modal dialogs, and dark mode support using CSS variables.

---

# 🚀 About The Project

Quick Notes is a beginner-friendly JavaScript CRUD application designed to practice real-world frontend development concepts without using any frameworks.

This project focuses on:

- JavaScript DOM Manipulation
- CRUD Operations
- Dynamic Rendering
- localStorage Data Persistence
- Modal/Dialog Handling
- Dark Mode Toggle
- Responsive UI Design
- Event Handling
- Template Literals
- JavaScript Arrays & Objects

The project was built as a practical learning project to improve JavaScript fundamentals and frontend development skills.
